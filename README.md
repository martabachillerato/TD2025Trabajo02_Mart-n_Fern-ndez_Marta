En este trabajo vamos a aplicar distintos métodos para decidir qué trabajo es mejor para un estudiante teniendo en cuenta 8 criterios diferentes. los archivos que están subidos son: 

- teoriadecision_funciones_multicriterio.R
- teoriadecision_funciones_multicriterio_diagrama.R
- teoriadecision_funciones_multicriterio_utiles.R
Son los 3 archivos proporcionados por los profesores que tienen guardados las distintas funciones que empleamos para realizar el trabajo.

- trabajo02.Rmd
Contiene todo el código del trabajo, junto a sus interpretaciones y los resultados.

- trabajo02.pdf
Vista en pdf del contenido del trabajo, junto a las interpretaciones y los resultados.

- ahptrabajo.ahp
Es el archivo de texto que he creado para aplicar el método AHP.

- tablatrabajo.png
- tablabtrabajo.png
Dos fotos exportadas del método AHP para tener los resultados de este visualmente.
